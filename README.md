# 🎓 Student Management System

## 📌 Overview

The **Student Management System** is a Python-based command-line application designed to store, manage, and analyze student records efficiently. The project demonstrates the practical implementation of fundamental Python concepts such as functions, loops, conditional statements, lists, dictionaries, input validation, and mathematical algorithms.

The system allows users to enter details for multiple students, including their name, roll number, age, and marks obtained in five subjects. After collecting the data, the program generates a detailed report showing each student's total marks, average score, grade, pass/fail status, highest and lowest marks, and several additional analyses.

This project emphasizes modular programming by dividing tasks into separate functions, making the code easy to understand, maintain, and reuse. Besides academic calculations, the system also includes mathematical and logical operations such as prime number checking, palindrome detection, Fibonacci series generation, and number manipulation.

Overall, the project serves as an excellent example for beginners to understand how various Python concepts can be combined to create a real-world application.

---

## ✨ Features

- ➤ Add and manage records of multiple students.
- ➤ Calculate total marks and average marks.
- ➤ Assign grades based on average scores.
- ➤ Determine pass or fail status.
- ➤ Categorize students by age group.
- ➤ Find highest and lowest marks.
- ➤ Check whether a roll number is a prime number.
- ➤ Detect whether a student's name is a palindrome.
- ➤ Reverse roll numbers using a while loop.
- ➤ Generate Fibonacci numbers.
- ➤ Display prime numbers within a given range.
- ➤ Find numbers divisible by both 3 and 5 between 100 and 200.
- ➤ Implement robust input validation.
- ➤ Modular and well-structured code.
